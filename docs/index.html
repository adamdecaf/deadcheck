Welcome to Deadcheck

Deadcheck is a dead man's switch service that works independently of its hosting infrastructure. By relying on external services for alerting, it ensures that critical tasks are handled even if the hosting server goes down.

Explore the sections below to learn how to set up Deadcheck using Docker, make HTTP check-ins, configure your alerts, and integrate with external services.
